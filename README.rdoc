# Gas Station Rater

Rails project that allows users to rate the quality of the service given by gas stations . It allows for user creation, administrators can setup all the nearby gas stations with their relevant information and users can comment and rate the service.

* This rating framework requires rails 4.1.6 and ruby 2.1.2p95.

* You will require the gems specified inside the gemfile.

* Simply download the framework, install the gems, do a rake db:create followed by a rake db:migrate and run the server with rails s.
